The objective of the game is to control the snake, maneuver it around obstacles (Asteroids),
collect space apples to grow in size, and avoid colliding with the walls, obstacles, or the snake's own body.

Game Mechanics:

1. Starting the Game:
- The snake will start with a length of one segment.
- Rocks (Asteroids) will be randomly placed on the game board.

2. Controls:
- Use the arrow keys (up, down, left, right) to control the direction of the snake.
- The snake will continuously move in the direction it is facing until the player changes its direction.

3. Snake Movement:
- The snake moves one grid square at a time.
- If the snake collides with the walls, obstacles (Asteroids), or itself, the game ends.

4. Collecting Apples:
- Space Apples will be randomly placed on the game board.
- When the snake eats a Space Apple, it grows in length by one segment.
- For each Space Apple eaten, the player earns points.

5. Obstacles (Asteroids):
- Rocks (Asteroids) will be scattered across the game board.
- The snake cannot pass through asteroids.
- If the snake collides with an asteroid, the game ends.

6. Game Over:
- The game ends when the snake collides with the walls, obstacles (Asteroids), or itself.
- A message will be displayed indicating the game is over, along with the player's score.
- The player will have the option to play again.
